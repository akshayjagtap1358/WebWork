📋 Instructions: Deploy with Data
Option A – localStorage (default, zero setup):
All data is stored in the browser. Works out of the box for single-device use.

Option B – Ship data with the site:
1. Click "Export Products JSON" above and save as data/products.json
2. Click "Export Settings JSON" and save as data/settings.json
3. Copy both files into assets/data.js as the VL_DEFAULT_PRODUCTS and VL_DEFAULT_SETTINGS variables (see README for format).

Option C – For multi-device / team use:
Deploy to a server and integrate a simple Node/PHP/Python backend that reads/writes JSON files. See README for a starter Express server.
